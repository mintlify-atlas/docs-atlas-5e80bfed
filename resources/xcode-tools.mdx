A curated collection of tools that enhance the Xcode development experience, from source editor extensions to standalone utilities and prototyping platforms.

## Xcode extensions (Xcode 8+)

Source editor extensions that add functionality directly inside Xcode.

<CardGroup cols={2}>
  <Card title="CleanClosureXcode" icon="code" href="https://github.com/BalestraPatrick/CleanClosureXcode">
    An Xcode Source Editor extension to clean the closure syntax.
  </Card>
  <Card title="xTextHandler" icon="code" href="https://github.com/cyanzhong/xTextHandler">
    Xcode Source Editor Extension Toolset — a collection of plugins for Xcode 8.
  </Card>
  <Card title="SwiftInitializerGenerator" icon="code" href="https://github.com/Bouke/SwiftInitializerGenerator">
    Xcode 8 Source Code Extension to generate Swift initializers automatically.
  </Card>
  <Card title="XcodeEquatableGenerator" icon="code" href="https://github.com/sergdort/XcodeEquatableGenerator">
    Xcode 8 extension to generate conformance to Swift Equatable protocol based on type and field selection.
  </Card>
  <Card title="Import" icon="package" href="https://github.com/markohlebar/Import">
    Xcode extension for adding imports from anywhere in the code.
  </Card>
  <Card title="Mark" icon="hash" href="https://github.com/velyan/Mark">
    Xcode extension for generating MARK comments.
  </Card>
  <Card title="XShared" icon="share" href="https://github.com/Otbivnoe/XShared">
    Xcode extension which copies code with special formatting quotes for social platforms (Slack, Telegram).
  </Card>
  <Card title="XGist" icon="github" href="https://github.com/Bunn/Xgist">
    Xcode extension to send your text selection or entire file to GitHub's Gist and copy the URL to the clipboard.
  </Card>
</CardGroup>

Additional extensions:

- [Swiftify](https://swiftify.com/) — Objective-C to Swift online code converter and Xcode extension.
- [DocumenterXcode](https://github.com/serhii-londar/DocumenterXcode) — Attempt to give a new life for VVDocumenter-Xcode as a source editor extension.
- [Snowonder](https://github.com/Karetski/Snowonder) — Magical import declarations formatter for Xcode.
- [XVim2](https://github.com/XVimProject/XVim2) — Vim key-bindings for Xcode 9.
- [Comment Spell Checker](https://github.com/velyan/Comment-Spell-Checker) — Xcode extension for spell checking and auto correcting code comments.
- [nef plugin](https://github.com/bow-swift/nef-plugin) — Xcode extension to export a code selection as a snippet. Available on the Mac App Store.

## Standalone iOS development tools

Command-line utilities and desktop applications that complement Xcode development.

<CardGroup cols={2}>
  <Card title="R.swift" icon="link" href="https://github.com/mac-cain13/R.swift">
    Get strongly typed, autocompleted resources like images, cells and segues in your Swift project.
  </Card>
  <Card title="SwiftGen" icon="bolt" href="https://github.com/SwiftGen/SwiftGen">
    A collection of Swift tools to generate Swift code for assets, storyboards, Localizable.strings, and UIColors.
  </Card>
  <Card title="Sourcery" icon="cpu" href="https://github.com/krzysztofzablocki/Sourcery">
    A tool that brings meta-programming to Swift, allowing code generation of Swift code.
  </Card>
  <Card title="XcodeGen" icon="hammer" href="https://github.com/yonaskolb/XcodeGen">
    Command line tool that generates your Xcode project from a spec file and folder structure.
  </Card>
  <Card title="Jazzy" icon="book" href="https://github.com/realm/jazzy">
    Soulful docs for Swift and Objective-C. Generates Apple-style documentation.
  </Card>
  <Card title="SwiftLint" icon="check-circle" href="https://github.com/realm/SwiftLint">
    A tool to enforce Swift style and conventions, integrated into Xcode builds.
  </Card>
  <Card title="Tuist" icon="layers" href="https://github.com/tuist/tuist">
    A tool to create, maintain and interact with Xcode projects at scale.
  </Card>
  <Card title="Xcodes.app" icon="download" href="https://github.com/RobotsAndPencils/XcodesApp">
    The easiest way to install and switch between multiple versions of Xcode.
  </Card>
</CardGroup>

Other notable tools:

- [Shark](https://github.com/kaandedeoglu/Shark) — Swift script that transforms the `.xcassets` folder into a type-safe enum.
- [Blade](https://github.com/jondot/blade) — Generate Xcode image catalogs for iOS/macOS app icons, universal images, and more.
- [Retini](https://github.com/terwanerik/Retini) — A super simple retina (2x, 3x) image converter.
- [Laurine](https://github.com/JiriTrecak/Laurine) — Localization code generator written in Swift.
- [BuildTimeAnalyzer](https://github.com/RobertGummesson/BuildTimeAnalyzer-for-Xcode) — Build Time Analyzer for Swift.
- [FengNiao](https://github.com/onevcat/FengNiao) — Command line tool for cleaning unused resources in Xcode.
- [LicensePlist](https://github.com/mono0926/LicensePlist) — A license list generator for all your dependencies in iOS applications.
- [Pecker](https://github.com/woshiccm/Pecker) — A tool to detect unused code in Swift projects.
- [SourceKitten](https://github.com/jpsim/SourceKitten) — An adorable little framework and command line tool for interacting with SourceKit.
- [xcbuild](https://github.com/facebook/xcbuild) — Xcode-compatible build tool from Facebook.
- [swiftenv](https://github.com/kylef/swiftenv) — Allows you to easily install and switch between multiple versions of Swift.
- [iSimulator](https://github.com/wigl/iSimulator) — GUI utility to control the Simulator and manage installed apps.
- [ProvisionQL](https://github.com/ealeksandrov/ProvisionQL) — Quick Look plugin for apps and provisioning profile files.
- [Swift Package Index](https://swiftpackageindex.com) — Swift packages list with information about quality and compatibility.
- [Rugby](https://github.com/swiftyfinch/Rugby) — Cache CocoaPods for faster rebuild and indexing of Xcode projects.
- [awesome-xcode-scripts](https://github.com/aashishtamsya/awesome-xcode-scripts) — A curated list of useful Xcode scripts.
- [Synx](https://github.com/venmo/synx) — A command-line tool that reorganizes your Xcode project folder to match your Xcode groups.
- [Xcode Template Manager](https://github.com/Camji55/xtm) — Swift command line tool that helps you manage your Xcode project templates.
- [Swift Macros](https://github.com/krzysztofzablocki/Swift-Macros) — A curated list of community-created Swift macros and learning resources.
- [Xcode Developer Disk Images](https://github.com/haikieu/xcode-developer-disk-image-all-platforms) — Disk images needed to deploy builds to devices when Xcode isn't updated.
- [Xcode Keymap for VS Code](https://marketplace.visualstudio.com/items?itemName=stevemoser.xcode-keybindings) — Extension that ports popular Xcode keyboard shortcuts to Visual Studio Code.
