## Fastlane and automation tools

[fastlane](https://github.com/fastlane/fastlane) is the most widely used iOS deployment automation tool. It connects all iOS deployment tools into one streamlined workflow.

```ruby Fastfile example
lane :beta do
  increment_build_number
  build_app(scheme: "MyApp")
  upload_to_testflight
  slack(message: "Successfully distributed a new beta build")
end

lane :release do
  capture_screenshots
  build_app(scheme: "MyApp")
  upload_to_app_store
end
```
