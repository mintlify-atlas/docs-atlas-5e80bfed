  <Accordion title="Camera & image capture">

    Frameworks for camera integration, image picking, barcode scanning, and custom AVFoundation camera views.

    <Note>
      YPImagePicker is one of the most popular image picker libraries with an Instagram-like UI. It supports photo libraries, camera capture, and video, with a clean Swift API.
    </Note>

    ```swift YPImagePicker example
    import YPImagePicker

    var config = YPImagePickerConfiguration()
    config.screens = [.library, .photo, .video]
    config.startOnScreen = .library

    let picker = YPImagePicker(configuration: config)
    picker.didFinishPicking { [unowned picker] items, cancelled in
        if let photo = items.singlePhoto {
            imageView.image = photo.image
        }
        picker.dismiss(animated: true)
    }
    present(picker, animated: true)
    ```
