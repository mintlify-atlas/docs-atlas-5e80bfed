Swift's `Codable` handles most JSON parsing, but these libraries fill gaps for complex scenarios, legacy formats, and richer type-safe querying.

```swift SwiftyJSON
import SwiftyJSON

let json = try JSON(data: responseData)
let name = json["user"]["name"].stringValue
let age = json["user"]["age"].intValue
let friends = json["user"]["friends"].arrayValue.map { $0["name"].stringValue }
```

```swift Codable (built-in)
struct User: Codable {
    let id: Int
    let name: String
    let email: String
}

let user = try JSONDecoder().decode(User.self, from: data)
```
