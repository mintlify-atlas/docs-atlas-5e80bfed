Reactive programming treats values as streams that change over time, making async code easier to compose, test, and reason about. This page covers the full spectrum — from RxSwift and Combine to functional programming utilities and event buses.

## RxSwift ecosystem

RxSwift is the Swift port of ReactiveX. It provides `Observable` sequences and a large suite of operators for transforming, filtering, and combining async streams.

```swift RxSwift example
import RxSwift
import RxCocoa

let disposeBag = DisposeBag()

searchBar.rx.text.orEmpty
    .debounce(.milliseconds(300), scheduler: MainScheduler.instance)
    .distinctUntilChanged()
    .flatMapLatest { query in
        API.search(query: query)
    }
    .bind(to: tableView.rx.items(cellIdentifier: "Cell")) { index, result, cell in
        cell.textLabel?.text = result.title
    }
    .disposed(by: disposeBag)
```
