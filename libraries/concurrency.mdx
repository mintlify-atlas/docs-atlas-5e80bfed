```swift Async (GCD syntax sugar) example
import Async

Async.background {
    // Runs on a background thread
    let data = fetchExpensiveData()
    return data
}.main { data in
    // Back on the main thread
    self.tableView.reloadData()
}
```

Full list of GCD utilities:

- [Async](https://github.com/duemunk/Async) — Syntactic sugar in Swift for asynchronous dispatches in Grand Central Dispatch.
- [Chronos-Swift](https://github.com/comyar/Chronos-Swift) — Grand Central Dispatch utilities including a repeating timer.
- [Dispatch](https://github.com/JARMourato/Dispatch) — Tiny library to make using GCD easier and intuitive.
- [GCDKit](https://github.com/JohnEstropia/GCDKit) — Grand Central Dispatch simplified with Swift.
- [GCDTimer](https://github.com/hemantasapkota/GCDTimer) — Well tested GCD Timer in Swift.
- [GrandSugarDispatch](https://github.com/jessesquires/GrandSugarDispatch) — Syntactic sugar for Grand Central Dispatch.
- [Me](https://github.com/pascalbros/Me) — A super slim solution to the nested asynchronous computations problem.
- [SwiftSafe](https://github.com/nodes-ios/SwiftSafe) — Thread synchronization made easy.
- [SwiftyTask](https://github.com/Albinzr/SwiftyTask) — An extreme queuing system with high performance for managing all tasks in an app with closures.
- [Threader](https://github.com/mitchtreece/Threader) — Pretty GCD calls and easier code execution.
- [YYDispatchQueuePool](https://github.com/ibireme/YYDispatchQueuePool) — iOS utility class to manage global dispatch queues and avoid thread explosion.
- [AlecrimAsyncKit](https://github.com/Alecrim/AlecrimAsyncKit) — Bringing async and await to Swift world with some flavouring (pre-Swift 5.5 async/await).

## Operation-based concurrency

Libraries built on top of `OperationQueue` for structured, cancellable task pipelines.

- [Queuer](https://github.com/FabrizioBrancati/Queuer) — A queue manager built on top of OperationQueue and Dispatch (aka GCD).
- [Overdrive](https://github.com/saidsikira/Overdrive) — Fast async task-based Swift framework with focus on type safety, concurrency, and multi-threading.
- [Kommander](https://github.com/intelygenz/Kommander-iOS) — Swift library to manage task execution in different threads using a Kommand concept.
- [Flow](https://github.com/JohnSundell/Flow) — Operation-oriented programming in Swift.
- [GroupWork](https://github.com/quanvo87/GroupWork) — Easy concurrent, asynchronous tasks in Swift.
- [SwiftQueue](https://github.com/lucas34/SwiftQueue) — Job scheduler with concurrent run, failure/retry, persistence, repeat, delay, and more.
- [Brisk](https://github.com/jmfieldman/Brisk) — A Swift DSL that allows concise and effective concurrency manipulation.
